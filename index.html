Instruções de pré-visualização
Para ver em 1920×1080 (apresentação real)
1. Abrir a tela num tab novo
2. Chrome/Edge: F12 → ícone de dispositivo Ctrl+Shift+M
3. Definir dimensão: 1920 × 1080 e escala 1×
4. Para captura: Ctrl+Shift+P → "Capture full size screenshot"
Para exportar PNG 4K (3840×2160)
1. Abrir DevTools → Ctrl+Shift+P
2. Escrever "device pixel" → definir Device Pixel Ratio = 2
3. Definir viewport para 1920×1080
4. Capturar → resultado: PNG 3840×2160
